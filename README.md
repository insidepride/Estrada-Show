# Estrada Show prototype

Статический мокап лендинга для кавер-группы Estrada Show.

## Что внутри

- `index.html` - структура лендинга.
- `styles.css` - визуальный стиль: темная сцена, розовый неон, блестки, адаптив.
- `app.js` - небольшое демо-поведение формы.
- `assets/` - оптимизированные изображения из предоставленных материалов.

## Просмотр

Откройте `index.html` напрямую в браузере или запустите локальный сервер:

```powershell
python -m http.server 8087 --bind 127.0.0.1
```

После запуска сайт доступен по адресу:

```text
http://127.0.0.1:8087/
```
